Google Sheets — Sync Real-time

Setiap update task langsung tercatat ke Google Sheets.
Update yang gagal (offline) otomatis dikirim ulang saat koneksi kembali.

Setup Google Sheets (lakukan sekali):
1. Buka script.google.com → New Project → paste kode Apps Script (lihat panduan) → ganti ID spreadsheet
2. Deploy → New Deployment → Web App → Execute as: Me → Who has access: Anyone
3. Copy URL → paste di kolom Script URL di bawah → klik Simpan
Sync Semua Data ke Google Sheets

Kirimkan seluruh data (projects, tasks, karyawan, update log, arsip) ke Google Sheets sekaligus.
Gunakan ini pertama kali setelah setup, atau setelah import data dari Excel.
